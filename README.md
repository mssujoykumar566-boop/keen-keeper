# 📌 KeenKeeper

---

## 📝 Description

**KeenKeeper** is a fully responsive friend interaction tracking web application. It allows users to track and manage interactions such as text messages, calls, and video calls with friends. The application provides a clean dashboard, timeline history, and statistical insights with data visualization.

---




## 🚀 Features

* **Fully Responsive Design**

  * Optimized for mobile, tablet, and desktop devices
  * Smooth and adaptive layout for all screen sizes

* **Fixed Navigation Bar**

  * Logo on the left side
  * Navigation links on the right: 
  🏠 Home ,⏳ Timeline ,📊 Stats

* **Banner Section**

  * Responsive hero section on the homepage

* **Dashboard Cards**

  * Total Friends
  * On Track
  * Need Attention
  * Interactions

* **Friends Section**

  * Displays a list of friends
  * Each friend includes tags for categorization
  * Clicking a friend shows detailed information

* **Interaction System**

  * Actions: Text, Call, Video
  * Each interaction is recorded and stored in the Timeline

* **Timeline Page**

  * Displays full interaction history
  * Supports sorting/filtering: Text, Call, Recent

* **Statistics Page**

  * Interactive pie chart using Recharts
  * Visualizes Text, Call, and Video interaction data

* **Loading State**

  * Spinner shown during data loading for better user experience
